# Mission Control (中文版)

Mission Control 是一个开源的 OpenClaw GUI 和 AI 智能体仪表盘，用于管理本地 AI 智能体、模型、记忆和技能。

## 功能特点

- **本地优先**：所有数据都保存在本地，保护您的隐私。
- **智能体管理**：轻松创建、配置和监控您的 AI 智能体。
- **模型管理**：支持多种本地和云端 LLM 模型。
- **定时任务**：自动化您的工作流程。
- **对话界面**：直接与您的智能体进行交互。
- **国际化支持**：现在已支持中文界面。

## 如何使用中文版

1. 在侧边栏底部找到语言切换按钮（地球图标）。
2. 点击按钮切换到“简体中文”。
3. 系统会自动保存您的语言偏好。
